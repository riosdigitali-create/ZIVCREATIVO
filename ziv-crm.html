    🔌 Conexión con Meta

    Tus credenciales se guardan solo en este navegador. Nunca salen de tu equipo.
    Cómo sacar tu token (2 minutos):
    1. Entra a business.facebook.com → Configuración del negocio
    2. Menú izquierdo → Usuarios → Usuarios del sistema → crea uno (rol: Administrador)
    3. Asígnale tu cuenta publicitaria con permiso de Administrar campañas
    4. Clic en Generar nuevo token → elige tu app → marca ads_read y read_insights
    5. Copia el token (empieza con EAA…) y pégalo arriba

    El token se guarda solo en este navegador. Nunca viaja a ningún servidor mío ni de nadie: el CRM le habla directo a Meta desde tu equipo.
